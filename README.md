## Running the Project.

```
yarn start
```

Then visit:

- The KeystoneJS Admin UI at [`http://localhost:3000/admin`](http://localhost:3000/admin)
- The GraphQL Playground at [`http://localhost:3000/admin/graphiql`](http://localhost:3000/admin/graphiql)
